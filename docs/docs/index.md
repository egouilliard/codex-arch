## What is Codex-Arch?

Codex-Arch is a comprehensive toolkit for analyzing and visualizing code architecture and dependencies. It helps developers understand complex codebases by extracting, analyzing, and documenting code structures.

### Key Features

- **File Tree Extraction**: Generate file tree representations for easy navigation
- **Dependency Analysis**: Track imports, dependencies, and relationships between modules
- **Metrics Collection**: Collect and visualize code statistics and complexity metrics
- **Graph Visualization**: Generate visual dependency graphs with customizable styles
- **Summary Generation**: Create high-level architecture summaries from code analysis
- **REST API**: Access all functionality programmatically through a RESTful interface
- **Git Integration**: Track code changes and update analysis incrementally
- **CLI Interface**: Powerful command-line tools for all operations

### Who Is It For?

- **Software Engineers** who want to understand large codebases quickly
- **Technical Leads** who need to document and communicate architecture
- **DevOps Teams** who want to automate code health tracking
- **New Team Members** who need to ramp up on project architecture
- **Open Source Maintainers** who want to provide clear documentation

## Getting Started

Explore the [installation guide](getting-started/installation.md) to set up Codex-Arch in your environment, then follow the [quick start guide](getting-started/quick-start.md) to begin analyzing your first project.

```bash
# Basic usage
codex-arch analyze /path/to/your/project
```
